This tool combines a curated wordbank with a small synthetic slogans dataset to build a lightweight, in‑browser language model (no server or tracking). You can replace or extend the dataset using the files in /data.

How it works (nerdy)
  1. We build a word‑level Markov chain from example slogans in slogans.csv (and any CSV you import).
  2. We graft templates using power words & industry terms to bias generations.
  3. We score candidates for punchiness (brevity, power words, rhythmic endings), then enforce constraints (keywords, avoids, rhyme/alliteration).
